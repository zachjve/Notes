## VSCode

#### ⌘f 
```
Faire une recherche par keyword dans une page.
```

#### ⌘s
```
Sauvegarder.
```

#### ⌘kc
```
Mettre en commentaire la selection.
```

#### ⌘ku
```
Décommenter la selection.
```

#### ⌘z
```
Retour en arrière.
```

#### ⌥ flèche haut/bas
```
Balader une ligne de haut en bas
```
